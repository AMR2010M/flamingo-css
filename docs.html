Flamingo CSS Documentation (CSS Features Only)

What is Flamingo CSS?

Flamingo CSS is a modern CSS framework based on attributes, making it easy to build beautiful, lively websites quickly and with minimal code.

Installation

Direct Download

<link rel="stylesheet" href="float.css">

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/amrkilany/flamingo-css@main/float.css">

Usage

Every feature in Flamingo CSS is an attribute you add directly to your element:

<div gradient-1 card center>Your content here</div>

You can combine multiple attributes on the same element:

<button btn="purple" shadow-3 pulse>Animated Button</button>
